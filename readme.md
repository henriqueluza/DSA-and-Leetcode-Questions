# LeetCode Solutions \- Python, SQL e Pandas

Repositório com resoluções de questões do LeetCode com explicações e código em Python, consultas SQL e soluções com Pandas.

## Sobre
Cada solução contém: enunciado resumido, abordagem, complexidade (tempo/espacial), código executável e exemplos\/testes.

## Estrutura do repositório
- `python/` \- soluções e scripts em Python
- `sql/` \- arquivos `.sql` com consultas e explicações
- `pandas/` \- notebooks e scripts com soluções usando Pandas
- `tests/` \- testes unitários (quando aplicável)
- `readme.md` \- este arquivo

## Formato das soluções
Cada problema deve seguir este padrão:
1. Título e link para a questão (LeetCode)
2. Enunciado resumido
3. Abordagem e raciocínio
4. Complexidade temporal e espacial
5. Código (com comentários)
6. Casos de teste / exemplos de entrada e saída

Exemplo de arquivo: `python/two_sum.py` ou `pandas/groupby_example.ipynb`

## Como usar
1. Clone o repositório:2. Para executar scripts Python:3. Para rodar testes (se houver):4. Para testar queries SQL, use seu SGBD preferido ou `sqlite3` para arquivos `.sql`.
